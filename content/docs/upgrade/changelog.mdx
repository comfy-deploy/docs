## v0.3.1 - August 5, 2025

### Added
- Support for ComfyUI v0.3.48

### Breaking Changes
- Workspace sessions require machine rebuild
- **Optimized Cold Start** now requires ComfyUI v0.3.45+

### Action Required
- **Update ComfyDeploy custom node** to latest version
- **Rebuild your machine** if unable to open workspace sessions
