Queue a new workflow run for a specific deployment with custom input parameters.

## Endpoint

```
POST /run/deployment/queue
```

## Request Body

```json
{
  "deployment_id": "deployment_67890",
  "inputs": {
    "input_name": "input_value",
    "another_input": "another_value"
  },
  "webhook_url": "https://your-app.com/webhook" // optional
}
```

## Response

Returns the created run information including the run ID and initial status.

```json
{
  "run_id": "run_12345",
  "status": "queued",
  "deployment_id": "deployment_67890",
  "created_at": "2024-01-15T10:30:00Z"
}
```
