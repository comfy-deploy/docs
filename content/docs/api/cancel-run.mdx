Cancel a workflow run that is currently queued or running.

## Endpoint

```
POST /run/{run_id}/cancel
```

## Parameters

- `run_id` (path, required): The unique identifier of the run to cancel

## Response

Returns confirmation of the cancellation request.

```json
{
  "run_id": "run_12345",
  "status": "cancelled",
  "cancelled_at": "2024-01-15T10:31:30Z",
  "message": "Run cancelled successfully"
}
```
